# Laminar_analysis_codes
This repository contains several scripts for preprocessing and layerfication of hippocampus @ 7 Tesla. The scripts call functions from multiple neuroimaging packages including SPM, FSL, ANTS and Hippunfold. 
It also uses the BM4D filter for denoising of the structural image, based on the work of Lüsebrink et al., 2021 and Maggioni et al.,2013. Acompcor (component based noise correction method) by Behzadi et al., 2007 is also used to further mitigate the noise in the fMRI data.
